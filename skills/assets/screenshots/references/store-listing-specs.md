# Store Listing Screenshot Specifications

## Google Play Store

### Screenshot Dimensions

| Device Type | Dimensions | Aspect Ratio | Required? |
|-------------|------------|--------------|-----------|
| Phone | 1080 x 1920 px (or 9:16) | Max 2:1 | Min 2 screenshots |
| 7" Tablet | 1200 x 1920 px | Max 2:1 | If supported |
| 10" Tablet | 1600 x 2560 px | Max 2:1 | If supported |

### Play Store Rules

- **Minimum**: 2 screenshots per device type
- **Maximum**: 8 screenshots per device type
- **Format**: JPEG or PNG
- **Transparency**: Not allowed
- **Max file size**: 8 MB per screenshot
- **Device frames**: Not allowed (Google adds them)
- **Text overlays**: Allowed but keep minimal

### Play Store Caption Limits

| Field | Max Length |
|-------|-----------|
| Short description | 80 characters |
| Long description | 4000 characters |
| Screenshot caption (short) | 80 characters |
| Screenshot caption (long) | 200 characters |

## Apple App Store

### Screenshot Dimensions

| Device | Display Size | Dimensions (Portrait) | Required? |
|--------|--------------|-----------------------|-----------|
| iPhone 16 Pro Max | 6.9" | 1320 x 2868 px | Required |
| iPhone 15 Pro Max | 6.7" | 1290 x 2796 px | Auto-scaled |
| iPhone 15 Plus | 6.5" | 1284 x 2778 px | Auto-scaled |
| iPhone 8 Plus | 5.5" | 1242 x 2208 px | Auto-scaled |
| iPad 12.9" / 13" | 12.9"–13" | See below | If iPad app |

#### iPad 12.9" / 13" Display Screenshots

Accepted dimensions (portrait or landscape):

| Dimensions | Orientation |
|------------|-------------|
| 2064 x 2752 px | Portrait |
| 2752 x 2064 px | Landscape |
| 2048 x 2732 px | Portrait |
| 2732 x 2048 px | Landscape |

- Up to **3 app previews** and **10 screenshots**
- A single set covers both 12.9" and 13" displays

### App Store Rules

- **Minimum**: 1 screenshot per device
- **Maximum**: 10 screenshots per device
- **App previews**: Up to 3 per device
- **Format**: JPEG or PNG
- **Transparency**: Not allowed
- **Max file size**: 8 MB per screenshot
- **First 3 screenshots**: Shown in search results (most important)

### App Store Caption Limits

| Field | Max Length |
|-------|-----------|
| Caption (overlay text) | 70 characters |
| Subtitle | 30 characters |
| Promotional text | 170 characters |
| Description | 4000 characters |

## Caption Best Practices by Screen Type

| Screen Type | Approach | Play Store Example | App Store Example |
|-------------|----------|--------------------|-------------------|
| Dashboard/Home | Highlight overview capability | "Everything you need, one screen" | "Your command center" |
| List/Feed | Emphasize discovery & browsing | "Browse and discover with ease" | "Find it fast" |
| Detail View | Focus on depth of information | "Dive deep into every detail" | "All the details" |
| Form/Input | Stress simplicity & speed | "Get started in seconds" | "Quick setup" |
| Chart/Analytics | Promote insight & tracking | "Turn data into actionable insights" | "Track your growth" |
| Settings/Profile | Highlight customization | "Tailor the experience to you" | "Make it yours" |
| Map/Location | Emphasize discovery | "Explore what's around you" | "Nearby & relevant" |
| Chat/Messaging | Focus on connection | "Stay connected, stay informed" | "Always in touch" |
| Media/Gallery | Highlight visual experience | "Your memories, beautifully organized" | "Stunning visuals" |
| Onboarding | Show ease of getting started | "Up and running in minutes" | "Easy to start" |

## Tone Guide by App Category

| Category | Tone Keywords | Do | Don't |
|----------|--------------|-----|-------|
| Business | Professional, efficient, streamlined | "Streamline your workflow" | "Cool tool for work stuff" |
| Health & Fitness | Encouraging, motivating, empowering | "Every step brings you closer" | "Track your boring workouts" |
| Education | Engaging, curious, empowering | "Master new skills daily" | "Study more efficiently" |
| Social | Warm, connected, expressive | "Share moments that matter" | "Post photos online" |
| Finance | Trustworthy, clear, secure | "Your finances, crystal clear" | "Money app" |
| Utility | Direct, practical, efficient | "Get it done in seconds" | "A useful tool" |
| Entertainment | Fun, exciting, immersive | "Endless entertainment awaits" | "Watch stuff" |
| Travel | Adventurous, inspiring | "Your next adventure starts here" | "Book trips" |
